You currently have **2 React applications** in your project:

---

## 🎯 **How to Run Your Professional React App**

### 1. Navigate to the correct directory:

```bash
cd Tourism-Management-System-main/tourism-react
```

### 2. Install dependencies (if not already done):

```bash
npm install
```

### 3. Start the development server:

```bash
npm start
```

### 4. Open your browser:

```
http://localhost:3000
```
